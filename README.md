# TemanTeman

Simple implementation of:
- RecyclerView (GridLayoutManager)
- SQLite
- SMS Manager


Hint:
- To send a message with AVD, you can input the phone number 5556 (from first AVD to second AVD) & 5554 (from second AVD to first AVD)
